Keyboard exercise II - recap

Highscore:

Copy, Cut, Paste

Goal

Arrange the jumbled text blocks in the correct order and duplicate certain blocks to create a structured sequence of 25 blocks in total. All actions shall be performed exclusively with keyboard commands.

Tasks

  1. Mark: Use the SHIFT key in combination with the arrow keys to select a text block.
  2. Cut and paste:
    • To cut a selected text block, press CTRL+X.
    • NUse the arrow keys to navigate to the position where the text block is to be inserted.
    • Press CTRL+V to insert the text block at the selected position.
  3. Copy and paste:
    • Some blocks must be duplicated. Select these blocks as described.
    • Press CTRL+C to copy the selected text block.
    • Navigate to the insertion point and press CTRL+V to insert a copy of the text block.

If all the blocks are arranged correctly, you can finish the test at the bottom of the page and see your time.
Do the test more often and try to become faster.
